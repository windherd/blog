广告名词解释
· 2018-11-13 · # code

1. 基本名词

CTR（Click Through Rate）：点击率，衡量线上广告效果的重要因素，等于广告点击数量与广告展现数量的比率，CTR越高，即点击广告的比率越高，广告效果越好。

CVR（Conversation Rate）：转化率，等于转化次数与到达次数的比率。这里的到达指用户点击进入广告主页面；转化指广告主所期望用户最终实现的行为，如购买、拨打电话等。转化率是效果广告最重要的指标之一。
